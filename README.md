# AI Period Tracker

An AI-powered menstrual health tracking app that:
- Logs symptoms
- Provides cycle insights
- Recommends health articles
- Shows cycle phases

## Tech Stack
- React.js
- Node.js 
- CSS

## Features
- Symptom logging
- Cycle tracking dashboard
- Insight panel with health tips
-  AI-style article recommendations
